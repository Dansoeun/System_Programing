## 작업 후 깃허브에 업로드 시 
1. ```
   git add *
   ```
2. ```
   git commit -m [message]
   ```
3. ```
   git push origin [브랜치명]
   ```

## 첫 브랜치 생성 시 
1. ```
   git clone https://github.com/Dansoeun/System_Programing.git
   ```
2. ```
   cd System_Programing
   ```
3. ```
   git checkout dev
   ```
4. ```
   git pull origin dev
   ```
5. ```
   git checkout -b [브랜치명]
   ```
6. ```
   git push origin [브랜치명]
   ```
